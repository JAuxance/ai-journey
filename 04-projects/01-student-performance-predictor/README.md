# Student Performance Predictor

## Objective
Predict student performance using tabular data and compare several machine learning models.

## Project structure
- `data/` dataset access and description
- `notebooks/` exploratory analysis
- `src/` reusable code
- `reports/figures/` visualizations
- `tests/` basic tests

## Workflow
1. Load data
2. Clean data
3. Explore variables
4. Prepare features
5. Train baseline and models
6. Evaluate results
7. Summarize findings

## Models
- Logistic Regression / Linear Regression
- Decision Tree
- Random Forest

## Metrics
- Accuracy / F1-score for classification
- RMSE / MAE for regression

## Key questions
- Which features matter most?
- Which model performs best?
- Where does the model fail?

## Conclusion
Summarize results honestly, including limitations and next improvements.
