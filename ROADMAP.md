# AI Journey Roadmap

## Phase 1 - Foundations for AI (Weeks 1-8)
### Objectives
- Become comfortable with Python for scientific computing
- Rebuild math foundations needed for ML
- Complete a first end-to-end ML project

### Focus
- Python: NumPy, pandas, matplotlib, scikit-learn
- Math: vectors, matrices, derivatives, averages, variance, probability basics
- Project: Student Performance Predictor

### Deliverables
- 3 to 5 Python mini-exercises per week
- 3 math note files per month
- 1 first ML project published

---

## Phase 2 - Classical Machine Learning (Months 3-6)
### Objectives
- Understand the main supervised and unsupervised algorithms
- Learn proper model evaluation and comparison

### Topics
- Linear regression
- Logistic regression
- KNN
- Decision trees
- Random forests
- SVM
- Clustering
- Feature engineering
- Cross-validation

### Deliverables
- 1 notebook or implementation per topic
- 2 portfolio-level tabular ML projects

---

## Phase 3 - Deep Learning (Months 6-12)
### Objectives
- Learn PyTorch correctly
- Understand neural networks from first principles
- Build image and sequence projects

### Topics
- Tensors and autograd
- MLPs
- CNNs
- RNNs / LSTMs
- Attention
- Transformers

### Deliverables
- PyTorch exercises
- 2 deep learning projects
- 1 model trained, evaluated, and documented cleanly

---

## Phase 4 - Research and Advanced Portfolio (Months 12-24)
### Objectives
- Reproduce papers
- Read research more comfortably
- Build an original project

### Topics
- Paper reading
- Reproduction methodology
- Experiment tracking
- Ablation mindset
- Writing technical reports

### Deliverables
- 1 polished research-style report
